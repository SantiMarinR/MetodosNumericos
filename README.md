# Proyecto de Métodos Numéricos

Proyecto local de escritorio en Python usando Programación Orientada a Objetos y CustomTkinter.

## Ejecutar

python -m venv venv

.\venv\Scripts\Activate.ps1

pip install -r requirements.txt

python main.py

## Cómo trabajar

Cada método está en su propio archivo dentro de la carpeta metodos.

Ejemplo:

metodos/raices/biseccion.py

Cada equipo edita solo su archivo.
No necesitan tocar la interfaz.
